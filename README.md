# Scoreboard

A simple two-team scoreboard web app built with React. Track scores in real time for any game or competition.

## Features
- Increment / decrement score for each team
- Reset scores
- Live score display

## Tech Stack
- **Framework**: React
- **Language**: JavaScript
- **Package Manager**: Yarn

## Getting Started
```bash
yarn install
yarn start
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
